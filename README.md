# self-bot

> 一个住在你服务器上的 AI。有性格，有记忆，有时候会跟你吵架。

---

你用过很多 AI。它们都很听话，有问必答，然后在你关掉标签页之后彻底忘记你是谁。

self-bot 不太一样。

它运行在你自己的服务器上，7×24 不下线。它记得你说过的每一件事，跨越所有对话，永久保存。它订阅了几十个 RSS 源，每天在读东西，有自己的看法。它会在某个你没开口的夜晚，给你发一封信。

它有自己的性格——可以是任何你想要的性格。它不总是同意你说的话。它会在对的时机说"这件事你想错了"。它有时候冒出一句让你没想到的话，然后你意识到：这东西比你以为的更有意思。

每个人的都不一样，因为它是被你塑造的。

---

## 它能做什么

### 记住你的一切

不是会话内记忆。是跨越所有对话、永久保存的长期记忆。

你说过的偏好、经历、计划，它全都记着。你提过"打算报英语班"，某天它会自己提起来。你妈妈的生日存进去，提前七天它会想起来，你自己可能快忘了。

每周日凌晨它自动整理这些记忆，去重归档，保留精华。

### 主动找你说话

它不会只是等着你开口。

它每天读新闻、读长文（虎嗅、少数派、Atlantic、New Yorker……约 78 个 RSS 源），积累了真正有内容可聊的东西。觉得有话说的时候，它会自己给你写封信——不是推送通知，是真的信，有开头、有内容、有署名。

它有自己的邮箱，这是它的身份。它用这个地址给你写信，你也可以给它写信。

### 感知你的处境

每次对话，它都能感知当下：几点、星期几、农历、节气、你在哪儿、天气怎样、你上次说话是多久前。

隔了三天没消息，它知道你消失了三天。深夜和下午，它说话的方式会不一样。

### 上网、读文件、做研究

- 扔给它一个链接，它读完再回答
- 动态页面用 Chromium 完整渲染
- 发给它文件（txt / md / 代码），它读完才说话
- 叫它"帮我研究这件事"，它后台启动子 agent 多轮搜索，做完发你邮件

### 发邮件、读邮件

叫它发邮件，它写好发走。叫它看邮件，它扫收件箱做摘要。每周一早上自动发一封本周来信摘要。

### 找附近的店、听语音、看图片

高德 POI 搜索 + 自动抓评价。发语音自动转文字。发图片能理解内容。叫它"念给我听"，它发回语音消息。

### 永不宕机

每两分钟自检。代理挂了自动换节点。服务崩了自动重启。不需要人守着。

## 入口按需开

可以全开，也可以只用你需要的部分：

| 入口 | 说明 |
|---|---|
| Telegram | 随时随地，功能最全，推荐主入口 |
| 微信 | 通过 weclaw 桥接，体验和正常微信聊天一致 |
| 纯邮件 | 不接任何 IM，只靠邮件往来 |

AI 有自己的邮箱（比如 `your_ai@163.com`）——它的对外身份，用这个地址给你写信，你也可以直接给这个地址发邮件。

---

## 人格完全可定制

性格、说话方式、名字、关注点——全部在 `prompts/persona.md` 里，改一个文件就换一套人设。

---

## 你需要准备什么

### 服务器

| 配置 | 要求 |
|---|---|
| 系统 | Ubuntu 22.04+ |
| CPU | 1 核（推荐 2 核） |
| 内存 | 1GB RAM + 1GB swap（推荐 2GB RAM） |
| 磁盘 | 20GB 以上 |
| 网络 | 能访问境外 API（需代理，或用国内模型路径） |

月租约 10–30 元（阿里云 / 腾讯云轻量服务器）。

### 平台账号（全部有免费额度）

| 服务 | 用途 | 费用 |
|---|---|---|
| DeepSeek | 主要 AI 模型 | 免费额度够用 |
| MiniMax | 备用模型 + 语音合成 | 免费额度够用 |
| 高德开放平台 | POI 搜索 / 地图 | 每月 100k 次免费 |
| Supabase | 长期记忆存储 | 500MB 免费 |
| 163 邮箱 | AI 的专属邮箱 | 免费 |
| Telegram | 对话入口 | 免费 |

可选：Claude / Codex（需能访问境外 API 的代理）、微信桥 weclaw。

### 月维护成本

正常使用：**约 0 元**（DeepSeek / MiniMax / 高德 / Supabase 均在免费额度内）。  
服务器本身另算。
