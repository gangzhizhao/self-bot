不是工具，
是它。

住在你服务器上的 AI。记得你说过的每一件事，读自己订阅的新闻，觉得有话说就给你写封信。每个人的都不一样，因为它是被你塑造的。

核心能力

🧠

长期记忆

不是会话内记忆。你说过的事、提过的计划、重要纪念日，全部永久保存，越用越懂你。

✉️

主动写信

它每天读几十个 RSS 源，有话想说的时候自己给你写封信。不是推送通知，是真的信。

⚡

有自己的性格

不总是同意你说的话。会顶嘴，会惊喜，有时冒出一句让你没想到的话。完全可定制。

🔒

你的服务器，你的数据

自托管，7×24 在线。数据在你自己的机器上，不上任何第三方云，自愈守护永不宕机。

🌐

能上网，能做研究

发链接它直接读，叫它调研一件事它后台多轮搜索，结果发你邮件。

🎙️

多模态

发语音自动转文字，发图片能理解内容，叫它念给你听它发回语音，还能收发邮件。

如何使用

技术用户

自己部署

代码完全开源，INSTALL.md 从零到跑起来约 40 分钟。你需要一台云服务器和几个免费 API key。

  • 1 核 1GB 服务器，约 15 元/月
  • DeepSeek / MiniMax / 高德 / Supabase 均有免费额度
  • Telegram 入口，官方 Bot API
  • 月运营成本约 0 元
非技术用户

找我搭建

不想折腾环境的，可以找我。我帮你配好一切，你只需要告诉我想要什么样的 AI。

  • 配置服务器环境 + 接 Telegram 入口
  • 给 AI 设专属邮箱和名字
  • 按你要求定制人格和说话方式
  • 📬
